# Capoeira Nota 10

Site institucional da escola Capoeira Nota 10, desenvolvido para presença digital, apresentação da escola, performance, SEO e comunicação direta com visitantes.

## Online

- Site: [capoeiranota10.com.br](https://capoeiranota10.com.br/)

## Objetivo

Construir uma página institucional rápida, responsiva e visualmente consistente para apresentar a escola, sua identidade, informações principais e canais de contato.

## Funcionalidades

- Página institucional responsiva
- Estrutura pensada para SEO
- Animações e interações leves
- Seções de apresentação, informações e chamada para contato
- Deploy em produção com domínio próprio

## Stack

- Vite
- TypeScript
- Tailwind CSS
- GSAP

## Estrutura

```text
.
├── public/       # Arquivos públicos
├── src/          # Código da interface
├── docs/         # Documentação auxiliar
├── index.html
└── package.json
```

## Como rodar localmente

```bash
npm install
npm run dev
```

Build de produção:

```bash
npm run build
```

Publicado e em manutenção.
